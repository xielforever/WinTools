# WinTools 进度跟踪

最后更新：2026-03-13

## 当前阶段

项目处于 **MVP 可用阶段（持续增强）**：主框架可运行，已完成文件管理核心能力与密码箱 V2（安全闭环 + 检索 + 密码质量），并已完成可下载即用的 GitHub Release 发布。

## 已完成

- 完成应用主框架（`main.py` + `wintools/app.py`）与模块化装载机制。
- 定义模块契约（`BaseModule`），并通过 `module_registry` 实现模块注册与导航切换。
- 实现目录大小统计模块（`modules/dir_size`）：
  - 支持选择目录并发起扫描。
  - 扫描在后台线程执行，UI 不阻塞。
  - 支持手动停止扫描（取消事件）。
  - 结果按“根目录 + 直接子目录”展示，每行大小按递归总大小计算。
  - 结果默认按大小降序。
  - 支持结果多标签页查看、右键关闭当前/其他标签页。
  - 支持右键“深入扫描此目录”“打开所在位置”“分析趋势”。
- 已接入 SQLite 持久化（`data/dir_size_history.db`）：
  - `scan_runs`、`scan_top_dirs` 表结构可自动初始化。
  - 支持旧结构兼容迁移（如 `path_key` 字段补齐）。
  - 每次扫描自动写入快照，可读取上次快照进行对比。
  - 支持目录历史趋势查询与可视化（弹窗折线图 + 明细表）。
- 已完成打包与发布基础链路：
  - 新增 `scripts/build.ps1`，一次构建同时产出 OneDir ZIP + OneFile EXE。
  - 新增版本接口 `wintools.__version__`（构建流程注入版本文件）。
  - 新增 GitHub Actions 发布工作流（Tag 自动发布 + 手动触发发布）。
  - Release 自动附带双产物，并使用自动 notes + 固定安装说明模板。
- 已完成线上发布验证（用户可下载并直接运行）：
  - 稳定版本：`v0.1.1`
  - 发布页：`https://github.com/xielforever/WinTools/releases/tag/v0.1.1`
  - 产物：
    - `WinTools-v0.1.1-windows-onedir.zip`
    - `WinTools-v0.1.1-windows-onefile.exe`
  - 历史版本 `v0.1.0` 已标记为 Deprecated（pre-release）并引导到 `v0.1.1`。
- 已完成密码箱模块 V2 改造（本地 SQLite，加密 blob）：
  - 安全闭环：
    - 会话自动锁定（1/5/10/30 分钟可配置），锁定后清空内存明文。
    - 编辑时仅当密码字段变更触发“覆盖旧密码”确认。
    - 复制密码倒计时提示与到期自动清空系统剪贴板（默认 30s）。
    - 状态栏不再暴露条目敏感上下文（不显示站点名）。
  - 检索与标签：
    - 即时搜索（站点/账号/备注）。
    - 标签筛选、仅看收藏、收藏切换。
    - 排序能力（更新时间/站点名/最近使用）。
  - 密码质量：
    - 新增/编辑支持密码生成器（长度与字符集可配置）。
    - 实时强度评估（弱/中/强）与弱密码确认提示。
    - 列表强度列，便于弱口令巡检。
  - 数据结构扩展：
    - 加密条目兼容新增字段：`tags/favorite/strength/last_used_at`。
    - 新增 `password_vault_settings` 表保存模块设置。
    - UI 非敏感状态持久化至 `data/ui_state.json`。
